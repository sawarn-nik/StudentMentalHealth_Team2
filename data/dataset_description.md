# Student Mental Health Survey Dataset

---

## Dataset Source
This dataset contains survey responses collected from university students to analyze factors affecting their mental health.

---

## Dataset Overview
The dataset includes demographic, academic, lifestyle, and psychological factors that may influence student mental health.

It is used to perform:
- Exploratory Data Analysis (EDA)
- Classification
- Clustering
- Association Rule Mining

---

| Feature | Value |

## Attribute Description

| Attribute | Description |
|----------|------------|
| Age | Age of the student |
| Course | Student's academic program |
| Gender | Gender of the student |
| CGPA | Cumulative Grade Point Average |
| Stress_Level | Stress level score (0–5) |
| Depression_Score | Depression score (0–5) |
| Anxiety_Score | Anxiety score (0–5) |
| Sleep_Quality | Quality of sleep (Poor / Average / Good) |
| Physical_Activity | Level of physical activity |
| Diet_Quality | Quality of diet |
| Social_Support | Level of social support |
| Relationship_Status | Relationship status of the student |
| Substance_Use | Substance use frequency |
| Counseling_Service_Use | Usage of counseling services |
| Family_History | Family history of mental illness |
| Chronic_Illness | Presence of chronic illness |
| Financial_Stress | Financial stress level (0–5) |
| Extracurricular_Involvement | Participation in extracurricular activities |
| Semester_Credit_Load | Number of academic credits taken in a semester |
| Residence_Type | On-campus or Off-campus residence |

## Purpose of the Dataset

This dataset is used to:

- Analyze factors affecting student mental health  
- Perform exploratory data analysis (EDA)  
- Build predictive models for mental health risk  
- Identify behavioral patterns using clustering  
- Discover relationships using association rule mining  

---

## How to Use the Dataset

1. Download the dataset file  
2. Place it inside the `data/` directory of the project  
3. Run the scripts in the `scripts/` folder to reproduce the analysis  

---

## Notes

- Scores such as Stress, Depression, Anxiety, and Financial Stress range from 0 to 5  
- Dataset is suitable for both supervised and unsupervised learning tasks
